# ipo-lr-12

## Задачи

- **Создать главное окно приложения** с функциональным меню и рабочей областью.
- Реализовать форму для добавления и редактирования клиентов с соответствующей валидацией полей.
- Реализовать форму для добавления и редактирования транспортных средств, учитывая специфические параметры для разных типов транспорта.
- Реализовать функциональность распределения грузов и отображения результатов в графическом интерфейсе.
- Добавить возможность сохранения и данных в файл.
- Обеспечить обработку ошибок и информирование пользователя через всплывающие сообщения или статусную строку.
- Добавить функции интерактивного взаимодействия (подсказки, сообщения об успешных действиях, обработка событий).
